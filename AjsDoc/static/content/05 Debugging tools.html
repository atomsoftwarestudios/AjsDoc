Debugging tools

Description

Debugging console is designed to be a modular console for multiple debugging tools such as the logger or the resource manager info.

Usage

Console can be configured to popup atuomatically after configured amount of time will elapse (i.e. 10s ater the page reload) or can be displayed manualy from the browser developer console by entering: ajs.debug.cosole.show() command.

The console can also be hidden by pressing the hide button in the console toolbar or by issuing the command in the developer console: ajs.debug.cosole.hide() command.
